== Description ==

Simple theme to customize WordPress admin. It is completely free.

== Installation ==

1. Upload `influid-admin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy!
